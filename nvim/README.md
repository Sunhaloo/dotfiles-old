# 💤 LazyVim

A starter template for [LazyVim](https://github.com/LazyVim/LazyVim).
Refer to the [documentation](https://lazyvim.github.io/installation) to get started.

---

If you are looking for my own neovim config; head over to [nvim_mine](https://www.github.com/Sunhaloo/dotfiles/tree/main/nvim_mine).

The plugins and configurations that we have in the LazyVim works for both Windows and Linux.
Nevertheless I do have some trouble with Termux ( *Terminal Emulator for Android* ). But it does work.
