# Alacritty Setup
This is my alacritty setup that can be used on most Operating System

## Requirements

- JetBrains Nerd Font Mono
- [Alacritty Themes Repository](https://github.com/alacritty/alacritty-theme)
    - You will have to clone this repository into `~/.config/alacritty/themes/` ( *just follow the guide* )
    - Hence you will have the most simple Alacritty Setup
