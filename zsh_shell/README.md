# Requirements
You are going to need to install / clone:

1. [eza](https://github.com/eza-community/eza)
2. [Fuzzy Finder ( fzf )](https://github.com/junegunn/fzf)
3. [zsh-syntax highlighting](https://github.com/zsh-users/zsh-syntax-highlighting/blob/master/INSTALL.md#in-your-zshrc)
4. [zsh-autosuggestions](https://github.com/zsh-users/zsh-autosuggestions/blob/master/INSTALL.md#oh-my-zsh)
